# Simple Background Remover

> A straightforward Python script dedicated to removing backgrounds from images automatically.

## ✨ Features
- Fast background extraction
- Supports multiple image formats

## 🚀 Getting Started
1. Clone the repository.
2. Install dependencies via pip (typically `rembg`, `pillow`).
3. Run the script passing your target image as an argument.
